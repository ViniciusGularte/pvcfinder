# PVC Finder Forge Mod

Forge workspace for Minecraft 1.21.10 using the official MDK.

Current features:

- `N` opens the in-game PVC Finder browser
- search over the bundled PVC shop snapshot
- `Track` button per offer
- in-world particle marker at the tracked shop
- marker only appears in the matching dimension
- marker clears itself once the player gets within 5 blocks

Notes:

- the bundled snapshot is copied from the root project's `data/shops-snapshot.json`
- Forge 1.21.10 targets Java 21
